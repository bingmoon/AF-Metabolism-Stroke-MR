# Metabolomic Profiling and Mendelian Randomization of Histidine Depletion in Atrial Fibrillation

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## Overview

This repository contains the complete analysis pipeline for the study:
**"Metabolomic Profiling Reveals Histidine Depletion in the Atrial Fibrillation Microenvironment and Its Genetic Association with Cardioembolic Stroke"**

We integrate:
- **Cellular metabolomics** (GC‑MS, Metabolomics Workbench ST004739)
- **Mendelian randomization** (Two‑sample MR using IEU OpenGWAS data)

to investigate the role of histidine metabolism in atrial fibrillation‑related cardioembolic stroke.

## Repository Structure

- `data/MSdata_ST004739_1.txt` – Raw metabolomic data (required input)
- `scripts/AF_Metabolism_MR_Pipeline.R` – Full analysis pipeline (Stages 1–5)
- `README.md` – This file
- `LICENSE` – Open source license

## Data Sources

- **Metabolomics**: Metabolomics Workbench Study ST004739 (DOI: 10.21228/M8085J)
- **GWAS summary statistics**: IEU OpenGWAS (https://gwas.mrcieu.ac.uk/)
  - Histidine: `met-a-495` (Shin et al., 2014)
  - Cardioembolic stroke: `ebi-a-GCST006908` (Malik et al., 2018)
  - C‑reactive protein: `ieu-b-35`
  - Heart failure: `ebi-a-GCST009541`

## Requirements

- R ≥ 4.3.1
- R packages: `TwoSampleMR`, `dplyr`, `ggplot2`, `impute`, `MRPRESSO`, `writexl`, `pacman`

## How to Run

1. Clone this repository.
2. Open `scripts/AF_Metabolism_MR_Pipeline.R` in RStudio or R console.
3. Set the working directory to the repository root.
4. Run the script. All figures and supplementary tables will be generated in the working directory.

The pipeline is self‑contained: it downloads sample metadata via the Metabolomics Workbench API, cleans and normalizes the raw data, performs differential analysis and pathway enrichment, conducts exploratory and targeted MR, and outputs publication‑quality figures and tables.

## Results

All figures (PDF) and supplementary tables (CSV/Excel) are automatically generated by the pipeline. Key outputs include:
- `Figure1_Volcano_Plot.pdf` & `Figure2_Heatmap.pdf` – Metabolomic profiling
- `Figure3_Pathway_Enrichment.pdf` – KEGG pathway enrichment bubble plot
- `Figure4_MR_Combined.pdf` – Mendelian randomization results
- `Supplementary_Tables_S1_S4.xlsx` – All supplementary tables
